# 1、 勇的释义 <!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)">

“勇”字在《论语》中一共出现了 16 次。根据《论语》的含义，勇主要可以分为三类：

1) 从积极意义上论述勇，与儒家推崇的“仁”、“义”、“礼”相关；
2) 好勇。从消极意义揭示“勇”的含义，表明“勇”与“利”和“智”的关系；
3) 君子之勇，孔子所追求的“勇”。

</grid>

---

<grid drag="100 12" drop="top" bg="white" border="5px solid yellow" style="border-radius:20px;" align="left">
# 2.1 知仁勇的关系 <!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft"  pad="5em">

“知、仁、勇”是孔子提倡和赞美的完美人格，并且孔子一生都在追求实践“知、仁、勇”这三者结合的完美人格。“知”通“智”，即智慧；“仁”是仁义，仁爱；“勇”是勇气，孔子曾把这三种人格比喻成不凋的松柏人格。 <!-- element animate="slideRightIn" frag="1"-->

子曰：“君子道者三，我无能焉：仁者不忧，知者不惑，勇者不惧。” <!-- element animate="slideRightIn" frag="2"-->

仁德的人不忧愁，智慧的人不迷惑，勇敢的人不惧怕。<!-- element animate="slideRightIn" frag="3" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em"-->



</grid>

---

<grid drag="100 12" drop="top" bg="white" border="5px solid yellow" style="border-radius:20px;" align="left">
# 2.2 勇与”义“的关系 <!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

子曰：见义不为，无勇也。（《论语·为政》） <!-- element animate="slideLeftIn"  frag="1"-->

勇与义相对，义是勇的基础，行勇之事的原则。<br><br>“知而不为，是无勇也。”（《朱熹集注》）<!-- element animate="slideRightIn" frag="2" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em"-->

子路曰：“君子尚勇乎？”子曰：“君子义以为上，君子有勇而无义为乱，小人有勇而无义为盗。”（《论语·阳货》）<!-- element animate="slideLeftIn"  frag="3"-->

义是勇的基础，对于君子来说，无义之勇则会造成混乱；对于小人来说，无义之勇则是因私自利，侵他人的利益。<!-- element animate="slideRightIn" frag="4" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em"-->

</grid>



---

<grid drag="100 12" drop="top" bg="white" border="5px solid yellow" style="border-radius:20px;" align="left">
# 2.3 勇与“礼”的关系 <!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

子曰：勇而无礼则乱，直而无礼则绞。（《论语·泰伯》）<!-- element animate="slideLeftIn"  frag="1"-->

勇缺乏义会引起祸乱，缺乏礼也会引起祸乱。<!-- element animate="slideRightIn" frag="2" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em"-->


子曰：“恶居下流而讪上者。恶勇而无礼者。恶果敢而窒者。”（《论语·阳货》）<!-- element animate="slideLeftIn"  frag="3"-->

超过义的范围，行为没有礼也就不能称为勇。<!-- element animate="slideRightIn" frag="4" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em"-->
</grid>

---

<grid drag="100 12" drop="top" bg="white" border="5px solid yellow" style="border-radius:20px;" align="left">
# 2.4 勇与“仁”的关系 <!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

子曰：仁者必有勇，勇者不必有仁。（《论语·宪问》）<!-- element animate="slideLeftIn"  frag="1"-->

+ 陆行而不避虎兕者，猎夫之勇也；<!-- element animate="slideLeftIn"  frag="2" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em; list-style: none;"-->
+ 水行不避蛟龙者，渔父之勇也；<!-- element animate="slideLeftIn"  frag="3" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em; list-style: none;"-->
+ 锋刃交于前，视若生者，烈士之勇也；<!-- element animate="slideLeftIn"  frag="4" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em; list-style: none;"-->
+ 知穷之有命，知通之有时，临大难而不者，仁者之勇也。<!-- element animate="slideLeftIn"  frag="5" style="border: 3px solid green; border-radius: 15px; padding: 0.8em; margin-left: 2em; list-style: none;"-->

由此可得，仁义礼勇的关系为：<!-- element animate="slideLeftIn"  frag="6"-->

君子之“勇”，以“义”为质，以“礼”为节，统摄于“仁”。<!-- element animate="slideLeftIn"  frag="7" style="border: 3px solid gray; border-radius: 15px; padding: 0.8em; margin-left: 2em"-->

</grid>

<grid drag="100 12" drop="top" bg="white" border="5px solid yellow" style="border-radius:20px;" align="left">
# 3、  好勇 <!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

子曰：道不行，乘桴浮于海，从我者其由与？子路闻之喜。子曰：由也好勇过我，无所取材。（《论语·公治长》）

> 孔子在这里提醒子路，其只是好勇而已，不能够审时度势，缺乏智慧的头脑。 <!-- element animate="fadeIn" frag="1" -->

子曰：好勇不好学，其蔽也乱。（《论语·阳货》） 

> 不学则不明其义，不究其实，以意会之，有转成不美者。（钱穆·《论语新解》）<!-- element animate="fadeIn" frag="2" -->

子曰：好勇疾贫，乱也。（《论语·泰伯》） 

> 惟主持治道，则须善体人情，导之以渐。一有偏激，乱起而祸且遍及于君子善人。（钱 穆·《论语新解》）<!-- element animate="fadeIn" frag="3" -->

</grid>

<grid drag="100 12" drop="top" bg="white" border="5px solid yellow" style="border-radius:20px;" align="left">
# 4、 君子之勇 <!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

> 子曰：知者不惑，仁者不忧，勇者不惧。（《论语·泰伯》）<br><br>
> 子曰：若臧武仲之知、公绰之不欲、卞庄子之勇、冉求艺，文之以礼乐，亦可以为成人矣。（《论语·宪问》）<br><br>
> 子曰：君子道者三，我无能焉。仁者不忧，知者不惑，勇者不惧。子贡曰：夫子自道也。（《论语·宪问》）<br><br>


明足以烛理，故不惑，理足以胜私，故不忧，气足以配道义，故不惧。此学之序也。（《四书集注》）

“君子之勇”就是在仁义礼智（知），乃至于利的框架下，儒家追求成圣成贤的动力。

</grid>

# 5.1 勇的实现 <!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

勇的内部“勇”是以“仁”为核心，“利”为补充，“智”为智慧，“义”为原则，“知”为理性，“礼”为规范，最终达到“君子之勇”的终极目标。

实现勇是为“有勇有谋”和“勇往直前”找到落脚之处，为价值的失落找到安身之所。

# 5.2 勇的实现：反省<!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

司马牛问君子，子曰：“君子不忧不惧。”曰：“不忧不惧，斯谓之君子已乎？”子曰：“内省不疚，夫何忧何惧？” （《论语·颜渊篇》）

> 这里的“忧”和“惧”，并不是指对外在的环境或东西的恐惧，而是与内在的自我反省相联系。若自我反省之后不惧于自己的良心，会更加坚定于自己的所作所为，不被外在的环境所困扰，就可以说是“勇”了。

勇者不惧代表着一种坚忍不拔的力量，一种为实现目标能够抵制外在的诱惑、克服欲望驱使的冲动、忍受简陋的生存环境的力量；一种寻求个人内心世界的平和、宁静、坦荡、冲破凡俗的力量。

</grid>

</grid>

<grid drag="100 12" drop="top" bg="white" border="5px solid yellow" style="border-radius:20px;" align="left">
# 5.2 勇的实现：修己<!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

### 我们要学习勇，实现勇，还要加强对”仁义礼……“的学习，加强对自己的修养。

“勇”一旦失去“礼”的约束，就会无视国法叛乱，可见徒有勇还不行，惟有遵循礼才会成为勇德。当然勇还需合乎“义”，“义者，宜也”，意思是做事要合乎事宜，凡事不可太过。所以要做到“勇”，就要“发乎仁，适乎礼，止乎义”，如此便可称得上“勇”。

“知、仁、勇”是孔子提倡和赞美的完美人格，并且孔子一生都在追求实践“知、仁、勇”这三者结合的完美人格。

> 子曰：“知、仁、勇三者，天下之达德也。所以行之者大。”（《论语·宪问》）

还需要做到真诚：
+ 不自欺，敢认错，“知耻近乎勇”

</grid>

# 5.3 勇的实现：磨砺<!-- element  style="margin-left: 2em"-->
</grid>

<grid drag="90 85" drop="5 12" bg="white" border="5px solid gray" style="border-radius:20px;font-size: 25px; font-weight: 700; color: rgb(96, 96, 117)" align="topleft" pad="5em">

### 子曰：“岁寒，然后知松柏之后凋也。”（《论语·子罕》）

苦难磨砺是生命最好的礼物。

> “只有经历过地狱般的磨练才能练出创造天堂的力量，只有流过血的手指才能弹出世间的绝唱。”（泰戈尔《飞鸟集》）

遇到困难时，首先应当是自己去解决，同时遇到问题时，也应当反思自己，而不是苛责别人。

“君子求诸己，小人求诸人。”（《论语·卫灵公篇》）
